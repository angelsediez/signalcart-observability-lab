# Screenshots

This directory stores visual evidence for SignalCart Observability Lab.

Screenshots are grouped by phase:

```text
assets/screenshots/
├── phase-00/
├── phase-01/
├── phase-02/
├── phase-03/
├── phase-04/
├── phase-05/
├── phase-06/
├── phase-07/
├── phase-08/
├── phase-09/
└── phase-10/
```

## Screenshot Rules

Screenshots should show useful operational evidence:

- service status
- health checks
- metrics
- Prometheus targets
- Grafana dashboards
- Alertmanager state
- incident signal
- recovery validation

Use clear names such as:

```text
P10-01-latency-spike-prometheus.png
P10-02-error-spike-grafana-api-red.png
P10-03-db-readiness-alert-prometheus.png
P10-04-recovery-validation-terminal.png
P10-05-final-compose-healthy.png
```
